Everything you need, one CLI

Install once globally or run via npx — zero config

$ npx cortex init Bootstrap .cortex/ in any project — detects your stack automatically
$ npx cortex compile Compile rules to all 9 AI provider formats at once
$ npx cortex score Analyze token budget, rule impact, and compression savings
$ npx cortex health Check model compatibility and provider health scores
$ npx cortex add react Install community rule packages for your stack
$ cortex switch claude Switch active AI provider and recompile context instantly
